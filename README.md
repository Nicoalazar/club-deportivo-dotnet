# Club Deportivo – Sistema de Gestión (.NET)

Sistema de gestión para un club deportivo, desarrollado en **C# / .NET** como proyecto colaborativo.

El objetivo del proyecto es modelar y resolver problemáticas reales de un club, como la administración de socios, actividades y lógica de negocio asociada.

## Conceptos trabajados

- Programación orientada a objetos
- Modelado de dominio
- Separación de responsabilidades
- Trabajo colaborativo con Git y GitHub
- Flujo de trabajo con branches y pull requests

## Tecnologías

- C#
- .NET
- Git / GitHub
