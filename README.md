# Pruebas C# – Desarrollo de Sistemas Orientados a Objetos

Repositorio grupal para las prácticas formativas y los trabajos prácticos de la materia **Desarrollo de Sistemas Orientados a Objetos**. Aquí iremos incorporando los distintos proyectos realizados durante el cursado, utilizando C# y .NET.

## Estructura

- `BibliotecaPrueba/`: biblioteca de clases compartida para ejercicios de manejo de colecciones y lectura de datos.
- `ClubDeportivo/`: solución orientada a objetos para la gestión de socios y actividades deportivas.
- `S1_Escribir_En_C#/`: ejercicios iniciales de programación en C# y sintaxis básica.
- `S2_A_Jugar_Con_Superheroes/`: práctica de clases, objetos y métodos.
- `S3_Registro_De_Postulantes/`: aplicación WinForms para carga y gestión de postulantes.
- `S3_Televisor.csproj`: modelado orientado a objetos de televisores y domiciliarios.
- `S4_Abstraer_Y_Encapsular/`: ejercicios sobre abstracción, encapsulamiento e interacción entre clases.
- `S4_Interface_De_Usuario/`: práctica de interfaces de usuario con WinForms (pantallas de login y formularios).
- `S6_Repaso/`: actividades de consolidación de contenidos previos.
- `S8_Herencia_Y_UML/`: prácticas avanzadas de herencia y diagramas UML.
- `ConsoleApp1.sln`: solución principal para abrir todos los proyectos desde Visual Studio.

## Requisitos

- .NET SDK (versión alineada con los `.csproj` de cada carpeta).
- Visual Studio o Visual Studio Code con extensiones para C#.

## Cómo empezar

1. Clona el repositorio.
2. Abre `ConsoleApp1.sln` o el proyecto que necesites en tu IDE.
3. Restaura paquetes y ejecuta el proyecto seleccionado.
